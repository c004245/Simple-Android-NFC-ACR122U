# Simple-Android-NFC-ACR122U
NFC Reader (ACR122u) module on Android

base https://www.acs.com.hk/en/driver/3/acr122u-usb-nfc-reader/

Two Acr122u NFC device, connect Android Tablet. and Try Reader CreditCard, NFC enabled phone.

so, The NFC unique number is then displayed in the TextView.

This NFC unique number is the same as the number when you turn on cell phone nfc mode.
